# Dialogs

Modal dialogs for settings, connections, session management, and other interactions.

## Dialog Base

All dialogs share common behavior:

```typescript
interface DialogProps {
  open: boolean;
  onClose: () => void;
  title: string;
  size?: 'sm' | 'md' | 'lg' | 'full';
}
```

**Sizes:**
| Size | Desktop Width | Mobile |
|------|--------------|--------|
| sm | 400px | 90% |
| md | 600px | 95% |
| lg | 800px | 100% |
| full | 100% | 100% |

**Common features:**
- Backdrop overlay (click to close, optional)
- Close button (Ã—) in header
- Escape key to close
- Focus trap within dialog
- Animated enter/exit
